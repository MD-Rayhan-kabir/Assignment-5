Important Faq

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, here are the key methods you can use: getElementById,getElementsByClassName,getElementsByTagName,querySelectorquerySelectorAll

What is event delegation in the context of the DOM?

Event delegation is a technique where a single event listener is attached to a parent element to handle events from its child elements. This is efficient because it utilizes event bubbling, allowing the parent to capture events from dynamically added or existing children.

How do you manipulate an element's attributes or styles using the DOM?

To manipulate an element's attributes in the DOM, you can use methods like `setAttribute()`, `getAttribute()`, or directly modify properties like `id`, `className`, etc. For styles, you can change the `style` property directly, such as `element.style.color = "red"` or `element.style.backgroundColor = "blue"`.

What is the purpose of the DOMContentLoaded event?

The `DOMContentLoaded` event triggers when the HTML is fully loaded and parsed, allowing scripts to run before external resources like images are loaded. It's used to initialize the DOM early.
